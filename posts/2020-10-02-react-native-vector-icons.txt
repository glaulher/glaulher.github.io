<p>O pacote de fonte de ícones, react-native-vector-icons, é uma forma de implementar icones no projeto de forma rápida e elegante, ícones presentes no pacote <a href="https://oblador.github.io/react-native-vector-icons/">clique aqui</a>{:target="_blank" rel="noopener"}. É ótimo para botões, logotipos, barras de navegação e guias, fácil de estender, estilizar e integrar no projeto.
Para utilizar os pacotes de fontes no React-native vamos utilizar a biblioteca react-native-vector-icons.</p>
<p>Criando um projeto para testes, considerando que tenha o react-native, yarn e emulador ja instalados, abra o terminal:</p>
<pre><code class="language-javascript">npx react-native init MyApp --template react-native-template-typescript
</code></pre>
<pre><code class="language-shell">cd MyApp
</code></pre>
<p>Instale a biblioteca de ícones:</p>
<pre><code class="language-shell">yarn add react-native-vector-icons
</code></pre>
<p>Realize o link da biblioteca com o código nativo com o comando:</p>
<pre><code class="language-shell">yarn react-native link
</code></pre>
<p>Carregue o projeto no emulador</p>
<pre><code class="language-shell">yarn android
yarn start
</code></pre>
<p>Será carregado o template padrão do react-native, para testar edite o App.tsx colando o código abaixo:</p>
<pre><code class="language-javascript">import React from 'react';
import {SafeAreaView, View, Text, StatusBar, StyleSheet} from 'react-native';
import Icon from 'react-native-vector-icons/Feather';

Icon.loadFont(); // carrega os icones

declare const global: {HermesInternal: null | {}};

const App = () => {
  return (
    &#x3C;>
      &#x3C;StatusBar barStyle="dark-content" />
      &#x3C;SafeAreaView>
        &#x3C;View style={styles.container}>
          &#x3C;Text>
            Olá
            &#x3C;Icon name="github" size={30} color="red" />
          &#x3C;/Text>
        &#x3C;/View>
      &#x3C;/SafeAreaView>
    &#x3C;/>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    alignItems: 'center',
    justifyContent: 'center',
    paddingTop: 200,
  },
});

export default App;

</code></pre>
