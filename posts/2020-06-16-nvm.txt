<p>O Node Version Manager é um script bash usado para gerenciar várias versões lançadas do Node.js. Plataformas: unix, macOS e windows WSL. Ele permite que você execute operações como instalar, desinstalar, alternar versão etc. Para instalar o nvm, use este script de instalação.</p>
<p>obs: é preciso ter o curl instalado.</p>
<p>no terminal:</p>
<pre><code class="language-shell">curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash
</code></pre>
<p>A execução de qualquer um dos comandos acima baixa um script e o executa. O script clona o repositório nvm para ~ / .nvm e tenta adicionar as linhas de origem do snippet abaixo ao arquivo de perfil correto(<code>~/.bash_profile</code>, <code>~/.zshrc</code>, <code>~/.profile</code>, or <code>~/.bashrc</code>).</p>
<pre><code class="language-shell">export NVM_DIR="$([ -z "${XDG_CONFIG_HOME-}" ] &#x26;&#x26; printf %s "${HOME}/.nvm" || printf %s "${XDG_CONFIG_HOME}/nvm")"
[ -s "$NVM_DIR/nvm.sh" ] &#x26;&#x26; \. "$NVM_DIR/nvm.sh" # This loads nvm
</code></pre>
<p>Para instalar uma versão do Node.js:</p>
<pre><code class="language-javascript">nvm install v14.3.0
</code></pre>
<p>Depois disso, você pode usar o nvm para alternar entre versões lançadas. Por exemplo, se a versão do Node.js que deseja utilizar for v14.3.0:</p>
<pre><code class="language-javascript">nvm use v14.3.0
</code></pre>
<p>Define a versão default do Node.js no shell:</p>
<pre><code class="language-javascript">nvm alias default v14.3.0
</code></pre>
<p>Desistalar uma versão do Node.js:</p>
<pre><code class="language-javascript">nvm uninstall 8.0.0
</code></pre>
<p>Listar versões instaladas no sistema:</p>
<pre><code class="language-shell">nvm list
</code></pre>
